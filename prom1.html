¿Qué aprenderemos?

En vía de lograr un plan de innovación a través de distintos recursos tecnológicos, en este promedio aprendemos sobre el desarrollo de una página web para presentar futuros proyectos tecnológicos innovadores.

Sesión 1

Hoy experimenté con la programación web usando VS Code. Comencé aprendiendo HTML, un lenguaje de marcado que utiliza etiquetas para estructurar páginas web. Las etiquetas usualmente tienen apertura y cierre, indicado por una barra diagonal.

  • Next

Sesión 2

En la primera clase aprendimos etiquetas esenciales, y en esta vimos más, aplicándolas en VS Code para ver cambios en la web de práctica. Cubrimos headers, párrafos, listas ordenadas y no ordenadas, y el concepto de anidación en el código. Además, obtuvimos un sitio template de HTML5 Up para estructurar nuestra bitácora. Sí, esta es la que están leyendo.

  • Next

Sesión 3

Durante la sesión, exploramos etiquetas HTML avanzadas para dar formato al texto, como negritas, cursivas, superíndices, subíndices, saltos de línea y reglas horizontales. También abordamos las diferencias en semántica y sintaxis del código HTML, y cómo estas afectan la apariencia de los sitios web.

  • Next

Sesión 4

En esta sesión, creamos subpáginas y ampliamos el espacio en el sitio web para más contenido de la bitácora. Implementamos botones de redirección, aprendimos a comentar el código para que no se muestre en la web y vinculamos nuestras redes sociales para que los visitantes puedan acceder a ellas fácilmente.

  • Next

Sesión 5

En esta clase, exploramos el diseño web y el poder de CSS para estilizar páginas. Descubrimos cómo modificar color, tipografía, tamaño y posición utilizando propiedades específicas para cada elemento.

  • Next

Sesión 6

En esta sesión, pusimos en práctica lo aprendido de CSS utilizando Visual Studio Code para visualizar cómo se reflejan los aspectos que deseamos. También exploramos nuevos atributos que podemos incorporar.

  • Next

Sesión 7

En esta clase, profundizamos en CSS: tamaños de caja, bordes, interacción con el puntero, celdas, posicionamiento, multicolumnas, y cuándo usar cada uno. También discutimos sobre hosting, dominios y cómo elegir un buen dominio para el sitio web.

  • Next

Sesión 8

Tras nuestra discusión previa sobre sitios web, en esta sesión creamos un enlace para nuestra página en Internet. Utilizamos Github y Netlify para lograrlo.

  • Next

Sesión 9

En esta sesión exploramos la electrónica básica con Tinkercad, identificando componentes esenciales para construir circuitos con Arduino. También profundizamos en la Ley de Ohm y comprendimos conceptos fundamentales sobre corrientes eléctricas.

  • Next

Sesión 10

En esta clase práctica, construimos circuitos desde cero usando el protoboard y Arduino. Programamos luces LED para realizar acciones específicas y exploramos circuitos en paralelo.
